Uber Rides Reviews

Purpose

Create a dataset of Uber Rides review data from Consumer Affairs and Sitejabber over the period of 2013 - 2019 containing comments, star-rating, and date.

Data Collection

ConsumerAffairs

Data was collected using the Python Scrapy web-crawling framework.

Sitejabber

Data was collected using the Python Beautiful Soup 4 library.

More Information

This dataset was collected for the larger purpose of conducting research on polarized ratings. Specifically, this project analyzes review data from Uber and AirBnB, companies that establish a connection between the consumer and service through direct contact, for trends in language usage in review comments. Overall, the findings suggest that there exists a polarity in the language used in consumer reviews, with the Uber data suggesting overly negative and positive language and the AirBnB data suggesting overly positive language.
