MC Scheduler

Purpose

A course schedule planning tool for Montgomery College students, enabling easy development of a schedule that fits their needs.

Features

  • Create a new schedule from scratch or select from a pre-existing schedule
  • Easy to use UI focused on keeping schedule making simple
  • Suggestive events visualizing potential time of classes and time conflicts
  • Up-to-date course information obtained directly from MC's catalog

How It Works

Data Collection

Montgomery College course catalog is webscraped by using Pythonâ€™s Selenium and BS4 packages. This data is stored as a JSON file.

Front End

The web-application uses the Vue JavaScript framework, as well as Vuetify and BootStrapVue componenets. Particularly, the visual display of the schedule is performed through Vuetify's calendar component.

Back End

Connecting to the front end by using axios AJAX requests, the Python-based back end runs on Flask and Flask Cors. Data is stored as JSON and accessed as Python objects.
